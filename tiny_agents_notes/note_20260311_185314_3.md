# 清理调试日志

## 问题
memory/types/semantic.py 中存在 13 处使用 `logger.info()` 输出调试日志，包含 emoji 标记。

## 影响
- 生产环境产生大量日志
- 影响性能
- 日志不专业

## 修复方案
```python
# 修复前
logger.info(f"🐛 [DEBUG] retrieve() 开始: query='{query}', limit={limit}")

# 修复后
logger.debug(f"retrieve() 开始: query='{query}', limit={limit}")
```

## 优先级
**中** - 影响性能和日志质量

## 验收标准
- 所有调试日志改为 debug 级别
- 移除 emoji 标记
- 添加配置开关控制日志级别
