# 修复裸 except 语句

## 问题
项目中多处使用裸 `except:` 语句，违反 PEP 8 规范，可能导致隐藏严重错误。

## 影响文件
1. tools/builtin/file_tools.py (2 处)
2. tools/builtin/note_tool.py (1 处)
3. memory/types/semantic.py (1 处)
4. memory/storage/qdrant_store.py (1 处)
5. memory/storage/neo4j_store.py (1 处)

## 修复示例
```python
# 修复前
except:
    pass

# 修复后
except (KeyError, ValueError, OSError) as e:
    logger.warning(f"操作失败: {e}")
```

## 优先级
**高** - 影响代码健壮性和可调试性

## 验收标准
- 所有裸 except 语句已替换为具体异常类型
- 异常信息被正确记录
- 测试通过
