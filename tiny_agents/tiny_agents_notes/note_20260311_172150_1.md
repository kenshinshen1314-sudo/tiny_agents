# tiny_agents 代码结构探索

## tiny_agents 项目结构分析

### 1. 核心架构
- **core/**: 核心基础组件
  - agent.py: Agent基类（43KB，核心抽象）
  - llm.py: LLM接口实现
  - message.py: 消息模型
  - config.py: 配置管理
  - lifecycle.py: 生命周期管理
  - session_store.py: 会话存储
  - streaming.py: 流式处理

### 2. Agent实现层
- **agents/**: 各类Agent实现
  - simple_agent.py: 简单Agent
  - react_agent.py: ReAct模式（推理+行动）
  - plan_solve_agent.py: 规划-求解模式
  - reflection_agent.py: 反思Agent
  - function_call_agent.py: 函数调用Agent（14KB，最新）

### 3. 工具系统
- **tools/**: 工具框架
  - base.py: 工具基类（14KB）
  - registry.py: 工具注册表（10KB）
  - chain.py: 工具链
  - circuit_breaker.py: 熔断器
  - builtin/**: 内置工具集
    - memory_tool.py: 记忆工具
    - note_tool.py: 笔记工具
    - search_tool.py: 搜索工具
    - file_tools.py: 文件操作
    - terminal_tool.py: 终端工具
    - rag_tool.py: RAG工具
    - calculator.py: 计算器

### 4. 记忆系统
- **memory/**: 多层次记忆管理
  - manager.py: 记忆管理器
  - base.py: 记忆基类
  - embedding.py: 向量嵌入
  - rag/**: RAG检索增强
  - storage/**: 存储适配器
  - types/**: 记忆类型
    - working: 工作记忆
    - episodic: 情景记忆
    - semantic: 语义记忆
    - perceptual: 感知记忆

### 5. 上下文管理
- **context/**: 上下文工程
  - builder.py: 上下文构建器
  - truncator.py: 截断器
  - token_counter.py: Token计数器

### 6. 可观测性
- **observability/**: 监控追踪
  - trace_logger.py: 追踪日志（16KB）

### 7. 技能系统
- **skills/**: 技能加载
  - loader.py: 技能加载器

### 8. 应用层
- **assistant/**: 助手应用
  - LearningAssistant: 学习助手
  - CodebaseMaintainerAssistant: 代码库维护助手

### 9. 演示与测试
- **demo/**: 22个演示脚本
- **tests/**: 完整测试套件

### 设计特点
1. 分层清晰：核心→实现→工具→应用
2. 模块化：各层职责明确，松耦合
3. 多Agent模式：ReAct、Plan-Solve、Reflection等
4. 完整工具生态：内置13+工具
5. 强记忆系统：4种记忆类型+RAG
6. 可观测性：完善的追踪机制
