# tiny_agents 项目代码结构分析

## 项目概览
tiny_agents 是一个轻量级 AI Agent 框架，版本 v0.1.0，实现了多种 Agent 及支持工具/函数调用。

## 核心模块结构

### 1. 核心层 (core/)
- **agent.py** (43KB) - Agent 基类，集成历史管理、工具截断、可观测性、技能加载
- **llm.py** (14KB) - LLM 接口封装
- **config.py** - 配置管理
- **message.py** - 消息定义
- **lifecycle.py** - 生命周期管理
- **session_store.py** - 会话存储
- **streaming.py** - 流式响应支持
- **database_config.py** - 数据库配置

### 2. Agent 实现 (agents/)
- **function_call_agent.py** (14KB) - 函数调用 Agent
- **react_agent.py** - ReAct 推理 Agent
- **plan_solve_agent.py** - 规划求解 Agent
- **reflection_agent.py** - 反思 Agent
- **simple_agent.py** - 简单 Agent
- 同时保留 _old 版本用于向后兼容

### 3. 工具系统 (tools/)
- **base.py** (14KB) - 工具基类
- **registry.py** (11KB) - 工具注册表
- **builtin/** - 内置工具集合:
  - memory_tool.py (19KB) - 记忆工具
  - note_tool.py (17KB) - 笔记工具
  - rag_tool.py (34KB) - RAG 检索工具
  - search_tool.py (21KB) - 搜索工具
  - terminal_tool.py (7KB) - 终端工具
  - file_tools.py (26KB) - 文件操作
  - calculator.py - 计算器
  - skill_tool.py - 技能工具
  - task_tool.py - 任务工具
  - todowrite_tool.py - TODO 工具
- **chain.py** - 工具链
- **circuit_breaker.py** - 熔断器
- **tool_filter.py** - 工具过滤

### 4. 上下文工程 (context/)
- **builder.py** (14KB) - 上下文构建器
- **history.py** - 历史管理
- **token_counter.py** - Token 计数
- **truncator.py** - 输出截断

### 5. 记忆系统 (memory/)
- **manager.py** (12KB) - 记忆管理器
- **base.py** - 记忆基类
- **embedding.py** (11KB) - 向量嵌入
- **storage/** - 存储后端:
  - qdrant_store.py (21KB) - Qdrant 向量库
  - neo4j_store.py (15KB) - Neo4j 图库
  - document_store.py (14KB) - 文档存储
- **rag/** - RAG 功能:
  - pipeline.py (43KB) - RAG 管道
  - document.py - 文档处理
- **types/** - 记忆类型定义
- **sessions/** - 会话存储
- **traces/** - 追踪记录

### 6. 可观测性 (observability/)
- **trace_logger.py** (16KB) - 追踪日志（JSONL + HTML）

### 7. 技能系统 (skills/)
- **loader.py** (6KB) - 技能加载器

### 8. 应用层 (assistant/)
- **LearningAssistant/** - 学习助手
  - PDFParserAssistant.py - PDF 解析助手
  - web/ - Web 前端
- **CodebaseMaintainerAssistant/** - 代码库维护助手
  - codebase_maintainer.py (16KB)

### 9. 演示与测试
- **demo/** (22个文件) - 各种功能演示脚本
- **tests/** (16个测试文件) - 单元测试

### 10. 文档与笔记
- **docs/** - 技术文档（调用链、代码执行流程）
- **tiny_agents_notes/** - 项目笔记
- MEMORY_FIX_SUMMARY.md, MEMORY_DEBUG_GUIDE.md - 记忆系统文档

## 架构特点
1. **模块化设计** - 核心功能分层清晰
2. **插件化工具** - 通过 ToolRegistry 管理工具
3. **记忆优先** - 完整的记忆系统（向量/图/文档）
4. **可观测性** - 完善的追踪和日志
5. **上下文工程** - Token 管理、历史压缩、输出截断
6. **向后兼容** - 保留旧版本 Agent
