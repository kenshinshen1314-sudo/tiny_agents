# tiny_agents 项目架构概览

## 项目基本信息
- 版本: v0.1.0
- 描述: 轻量级 AI Agent 框架
- 路径: /Users/kenshin/Projects/my-first-agent/tiny_agents

## 核心目录结构

### 1. 核心模块 (core/)
- agent.py: Agent 基类，包含完整的生命周期管理
- llm.py: LLM 接口封装
- config.py: 配置管理（支持环境变量）
- message.py: 消息模型
- session_store.py: 会话持久化
- lifecycle.py: 生命周期事件系统
- streaming.py: 流式输出支持
- database_config.py: 数据库配置

### 2. Agent 实现 (agents/)
- simple_agent.py: 简单对话 Agent
- react_agent.py: ReAct 推理 Agent
- plan_solve_agent.py: 规划-求解 Agent
- reflection_agent.py: 反思型 Agent
- factory.py: Agent 工厂（支持子代理机制）

### 3. 工具系统 (tools/)
- base.py: 工具基类
- registry.py: 工具注册表
- async_tool_executor.py: 异步工具执行器
- circuit_breaker.py: 熔断器
- chain.py: 工具链
- tool_filter.py: 工具过滤器

### 4. 内置工具 (tools/builtin/)
- search.py: 搜索工具
- calculator.py: 计算器
- file_tools.py: 文件操作
- memory_tool.py: 记忆工具
- note_tool.py: 笔记工具
- rag_tool.py: RAG 工具
- task_tool.py: 子代理任务工具
- todowrite_tool.py: 进度管理
- devlog_tool.py: 开发日志
- skill_tool.py: 知识外化
- terminal_tool.py: 终端工具

### 5. 上下文工程 (context/)
- history.py: 历史管理（压缩策略）
- truncator.py: 工具输出截断
- token_counter.py: Token 计数器
- builder.py: 上下文构建器

### 6. 记忆系统 (memory/)
- base.py: 记忆基类
- manager.py: 记忆管理器
- embedding.py: 嵌入生成
- rag/: RAG 实现
- sessions/: 会话存储
- storage/: 持久化存储
- traces/: 追踪日志

### 7. 可观测性 (observability/)
- trace_logger.py: 追踪日志（JSONL + HTML）
- metrics.py: 指标收集

### 8. 技能系统 (skills/)
- skill_loader.py: 技能加载器
- 支持知识外化

### 9. 演示程序 (demo/)
包含 20+ 演示程序，展示各种功能

### 10. 测试 (tests/)
完整的测试套件

### 11. 文档 (docs/)
包含测试流程文档

## 关键特性

### 1. 多种 Agent 类型
- SimpleAgent: 基础对话
- ReActAgent: 推理-行动循环
- PlanSolveAgent: 规划-求解模式
- ReflectionAgent: 反思型学习

### 2. 完整的工具生态
- 工具注册表
- 异步执行
- 熔断器保护
- 工具链组合
- 工具过滤

### 3. 上下文管理
- 历史压缩（简单摘要/智能摘要）
- 工具输出截断
- Token 计数与缓存

### 4. 可观测性
- JSONL 追踪日志
- HTML 可视化报告
- 会话持久化

### 5. 高级特性
- 子代理机制（上下文隔离）
- Skills 知识外化
- TodoWrite 进度管理
- DevLog 开发日志
- 流式输出
- 异步生命周期钩子

## 配置要点

核心配置项（Config 类）：
- context_window: 128000 tokens
- compression_threshold: 0.8
- trace_enabled: True
- skills_enabled: True
- subagent_enabled: True
- session_enabled: True
