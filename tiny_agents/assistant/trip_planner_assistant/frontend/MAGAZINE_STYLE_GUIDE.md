# 旅行杂志风格 - 设计指南

## 概述

这是一套专业的"旅行杂志风格"前端设计，旨在为用户提供优雅、大气的旅行规划体验。

## 设计特点

### 🎨 视觉风格
- **字体系统**：
  - 标题：Playfair Display（优雅衬线字体）
  - 副标题：Cormorant Garamond（精致衬线字体）
  - 正文：Lato（现代无衬线字体）

- **色彩方案**：
  - 金色主色：#D4A574
  - 深棕辅色：#2C2C2C
  - 米白背景：#FAF8F5
  - 陶土红强调：#C17A5B

- **布局特点**：
  - 大胆的非对称排版
  - 充足的留白空间
  - 装饰性线条和边框
  - 纸张纹理叠加效果

### ✨ 特色元素

1. **装饰性背景**
   - 纸张纹理叠加
   - 垂直装饰线条
   - 渐变色彩边框

2. **优雅的排版**
   - 杂志风格的大标题
   - 装饰性符号和分隔线
   - 编号标签系统

3. **交互动画**
   - 流畅的过渡效果
   - 悬停状态反馈
   - 加载动画

## 使用方式

### 路由访问

有两种风格可供选择：

1. **现代风格**（默认）
   - 首页：`http://localhost:5173/`
   - 结果页：`http://localhost:5173/result`

2. **杂志风格**
   - 首页：`http://localhost:5173/magazine`
   - 结果页：`http://localhost:5173/magazine/result`

### 风格切换

在页面顶部的导航栏中，点击"现代"或"杂志风"按钮即可切换风格。

## 自定义配置

### 修改颜色方案

在组件的 `<style scoped>` 部分修改 CSS 变量：

```css
:root {
  --color-gold: #D4A574;        /* 主金色 */
  --color-gold-light: #E8C9A0;   /* 浅金色 */
  --color-brown: #2C2C2C;        /* 深棕色 */
  --color-cream: #FAF8F5;        /* 米白色 */
  --color-terracotta: #C17A5B;   /* 陶土红 */
}
```

### 修改字体

更换 Google Fonts 链接：

```html
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&family=Lato:wght@300;400;700&display=swap');
```

然后在 CSS 变量中更新：

```css
:root {
  --font-title: 'Your Title Font', serif;
  --font-body: 'Your Body Font', sans-serif;
}
```

## 响应式设计

两个页面都完全响应式，支持以下断点：

- **桌面**：> 1200px - 完整的双栏布局
- **平板**：768px - 1200px - 调整后的布局
- **移动**：< 768px - 单栏堆叠布局

## 打印支持

结果页支持打印功能，点击"保存为PDF"或使用浏览器打印功能可生成优化的打印版本。

## 性能优化
