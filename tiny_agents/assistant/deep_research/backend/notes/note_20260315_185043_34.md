# 任务 1: 基础背景梳理 - OpenClaw助力OPC一人公司研究

**任务概述**：深度研究OpenClaw助力OPC(一人公司)的几个方向

**信息来源**（5篇）：
1. 新浪新闻：OpenClaw助力一人公司兴起，普通人如何抓住AI创业风口？
2. 新华网：AI时代，"一人公司"加速孵化
3. 腾讯云开发者社区：辩证看待养龙虾（OpenClaw）与OPC一人公司
4. CSDN博客：深圳龙岗发布《支持OpenClaw&OPC发展若干措施》征求意见稿
5. 知乎：OpenClaw 多智能体系统深度技术解析

**核心概念界定**：
- OpenClaw：能直接操控电脑执行任务的AI智能体，用户养成过程戏称为"养龙虾"
- OPC（One Person Company）：一人公司，借助AI智能体实现"单人驱动+AI协同"的创业新范式

**关键发现**（5条）：
1. OpenClaw定位独特：非开发者工具库，而是即用型智能体产品，可直接操控电脑执行任务
2. 政策积极响应：北京发布OPC服务计划，深圳龙岗"龙虾十条"，多地布局OPC创业生态
3. 三大落地路径：垂直领域服务、工具化技能定制、自动化流程设计
4. 核心风险警示：部署门槛、安全隐患、能力错配（4.2万实例遭黑客攻击案例）
5. 辩证视角强调：智能体放大能力而非创造能力，OPC降低成本而非消灭成本

**时间背景**：2026年3月，正好是人形机器人量产元年及具身智能爆发期

**后续研究方向建议**：
- 技术原理深度解析
- 具体应用场景案例
- 政策详细对比分析
- 风险防控机制研究
